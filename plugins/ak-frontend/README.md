# ak-frontend

Frontend development plugin for AgentKit with specialized agents and pattern knowledge.

## Agents

- **frontend-specialist** — Modern JavaScript frameworks, component architecture, UX optimization
- **tailwind-css-specialist** — Tailwind CSS v4 best practices, configuration, integration

## Knowledge Base

- `accordion.md` — Accordion UI pattern implementation
- `alpine-js-integration.md` — Alpine.js with TYPO3 Content Blocks
- `tailwind-patterns.md` — Tailwind CSS patterns and optimization strategies

## Usage

Agents are invoked automatically through core skills or directly via Task tool:

```
Use Task tool with subagent_type="frontend-specialist"
Use Task tool with subagent_type="tailwind-css-specialist"
```
