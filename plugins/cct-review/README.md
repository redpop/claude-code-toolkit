# cct-review

Code review integration plugin for Claude Code Toolkit.

## Skills

| Skill | Description |
|-------|-------------|
| `coderabbit` | CodeRabbit CLI integration for automated code review with critical evaluation |

## Usage

```bash
/cct-review:coderabbit
/cct-review:coderabbit --uncommitted
/cct-review:coderabbit --pr 123
```

## Requirements

- CodeRabbit CLI (`npm install -g coderabbit`)
