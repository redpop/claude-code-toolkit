# ak-typo3

TYPO3 CMS development plugin for AgentKit. Comprehensive tooling for TYPO3 v13.4 development.

## Skills

| Skill | Description |
|-------|-------------|
| `content-blocks` | Generate Content Blocks v1.3 with modern field configurations |
| `sitepackage` | Create TYPO3 v13.4 SitePackage with Site Sets |
| `extension-kickstarter` | Create extensions using ext-kickstarter or manual scaffolding |
| `fluid-components` | Generate Fluid v4 Components with Atomic Design patterns |
| `make-content-block` | Wrapper for native make:content-block with intelligent defaults |

## Agents

- **typo3-architect** — Enterprise CMS architecture, performance, security
- **typo3-content-blocks-specialist** — Content Blocks v1.3 configuration and optimization
- **typo3-extension-developer** — Extension development, Extbase, services, events
- **typo3-fluid-expert** — Fluid template analysis and ViewHelper development
- **typo3-typoscript-expert** — TypoScript, Site Sets, data processing

## Knowledge Base

15+ reference files covering sitepackage configuration, Content Blocks patterns, troubleshooting, CLI commands, field naming conventions, and more.

## Usage

```bash
/ak-typo3:content-blocks hero-section --type=element --fields=title,text,image
/ak-typo3:sitepackage mycompany corporate-site
/ak-typo3:extension-kickstarter my_extension --type=plugin
/ak-typo3:fluid-components card --type=molecule
/ak-typo3:make-content-block --vendor=mycompany
```
