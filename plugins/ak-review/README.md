# ak-review

Code review integration plugin for AgentKit.

## Skills

| Skill | Description |
|-------|-------------|
| `coderabbit` | CodeRabbit CLI integration for automated code review with critical evaluation |

## Usage

```bash
/ak-review:coderabbit
/ak-review:coderabbit --uncommitted
/ak-review:coderabbit --pr 123
```

## Requirements

- CodeRabbit CLI (`npm install -g coderabbit`)
