# cct-git

Git operations and workflow management plugin for Claude Code Toolkit.

## Skills

| Skill | Description |
|-------|-------------|
| `operations` | Smart Git operations with scope detection and commit pattern learning |

## Agents

- **git-workflow-specialist** — Intelligent commit creation, change analysis, professional messaging
- **git-conflict-specialist** — Conflict analysis, resolution strategies, merge semantics

## Usage

```bash
/cct-git:operations commit --smart
/cct-git:operations review --uncommitted
/cct-git:operations resolve
```
